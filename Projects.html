Parksense is an ongoing project that combines Python, OpenCV, and Raspberry Pi to create a smart car parking detection system. Tiny Tracker

Tiny Tracker is a web application designed to simplify time sheet management for childcare providers. It was developed using Python, Flask, SQLite, SQLAlchemy, HTML, CSS (Bootstrap), and limited JavaScript. The project features include clocking in and out, adding and deleting children, and generating monthly reports.
